# Раскладка клавиатуры для османской транскрипции

Раскладка клавиатуры для **Linux** и **macOS**, позволяющая легко вводить диакритические знаки османско-турецкой транскрипции: ḏ, ā, ḍ, ġ, ḫ, ṣ, ṭ, ẓ, ʾ, ʿ и другие.

Основана на турецкой раскладке Q. Символы транскрипции доступны через клавишу **AltGr** (Linux) или **Option/⌥** (macOS).

> **Что такое AltGr?** Это клавиша Alt на правой стороне клавиатуры. На некоторых клавиатурах обозначена как «Alt Gr» или просто «Alt».

## Карта клавиш

### Прямой доступ (AltGr / Option + буква)

| Клавиша | Строчная | Прописная | Описание |
|---------|----------|-----------|----------|
| AltGr+A | ā | Ā | a с макроном |
| AltGr+D | ḏ | Ḏ | d с чертой снизу |
| AltGr+E | ē | Ē | e с макроном |
| AltGr+G | ġ | Ġ | g с точкой сверху |
| AltGr+H | ḫ | Ḫ | h с бревисом снизу |
| AltGr+I | ī | Ī | i с макроном |
| AltGr+K | ḳ | Ḳ | k с точкой снизу |
| AltGr+O | ō | Ō | o с макроном |
| AltGr+S | ṣ | Ṣ | s с точкой снизу |
| AltGr+T | ṭ | Ṭ | t с точкой снизу |
| AltGr+U | ū | Ū | u с макроном |
| AltGr+N | ʾ | ʾ | хамза |
| AltGr+M | ʿ | ʿ | айн |

> **Для заглавных букв:** удерживайте Shift при нажатии AltGr + буква. Пример: Shift+AltGr+A = Ā

### Мёртвые клавиши (AltGr / Option + клавиша, затем буква)

Мёртвые клавиши работают в два шага:
1. Нажмите **AltGr + клавишу-модификатор** и **отпустите** (на экране ничего не появится — это нормально)
2. Затем нажмите нужную **букву**

> **Пример:** чтобы ввести ḏ → нажмите AltGr+Z и отпустите, затем нажмите d.

| Клавиша | Модификатор | Использование |
|---------|-------------|---------------|
| AltGr+Z | черта снизу | затем d→ḏ, z→ẕ, t→ṯ, s→s̱ |
| AltGr+X | макрон | затем a→ā, u→ū, i→ī, e→ē, o→ō |
| AltGr+C | точка снизу | затем d→ḍ, z→ẓ, k→ḳ, h→ḥ, t→ṭ, s→ṣ |
| AltGr+V | точка сверху | затем g→ġ, z→ż |
| AltGr+B | бревис снизу | затем h→ḫ |

## Простая установка (рекомендуется)

Скачайте установщик со страницы [Releases](https://github.com/podeveloper/osmanlica-transkripsiyon-klavye/releases/latest):

### Linux (Ubuntu / Debian)

1. Скачайте файл `.deb` со страницы [Releases](https://github.com/podeveloper/osmanlica-transkripsiyon-klavye/releases/latest)
2. **Дважды щёлкните** по файлу и нажмите «Установить»
   - Или через терминал: `sudo dpkg -i osmanlica-klavye_*.deb`
3. Включите раскладку:

**GNOME (Ubuntu по умолчанию):**
- Настройки → Клавиатура → Источники ввода → **+** → В разделе «Турецкий» выберите **«Turkish (Ottoman Transcription Q)»**

**KDE (Kubuntu):**
- Параметры системы → Устройства ввода → Клавиатура → Раскладки → Добавить → Вариант: Ottoman Transcription

4. Переключайтесь между раскладками с помощью значка клавиатуры в панели
   - Горячая клавиша: **Super+Space** (GNOME) или **Meta+Space** (KDE)

Для удаления: `sudo apt remove osmanlica-klavye`

> **Пользователи Wayland:** Ubuntu 22.04+ по умолчанию использует Wayland. Команда `setxkbmap` не работает в Wayland. Используйте способ через Настройки. Чтобы переключиться на X11, выберите «Ubuntu on Xorg» на экране входа.

### macOS

1. Скачайте файл `.pkg` со страницы [Releases](https://github.com/podeveloper/osmanlica-transkripsiyon-klavye/releases/latest)
2. **Дважды щёлкните** по файлу и следуйте инструкциям установщика
3. **Выйдите из системы и войдите снова** (этот шаг обязателен)
4. **Системные настройки** → **Клавиатура** → **Источники ввода** → **+** → Выберите **«Turkish - Ottoman Transcription Q»** → **Добавить**
5. Переключайтесь между раскладками через значок клавиатуры в строке меню
   - Горячая клавиша: **Ctrl+Space** или **fn+Space**

> На macOS вместо AltGr используется клавиша **Option (⌥)**. Она находится рядом с клавишей Command.

## Проверка установки

1. Откройте любой текстовый редактор (LibreOffice Writer, TextEdit, Блокнот и т.д.)
2. Выберите раскладку «Ottoman Transcription»
3. Нажмите **AltGr+A** → должен появиться символ **ā**
4. Нажмите **AltGr+N** → должен появиться символ **ʾ** (хамза)
5. Нажмите AltGr+Z и отпустите, затем нажмите **d** → должен появиться символ **ḏ**

Если все символы отображаются правильно, установка прошла успешно!

> **Если вместо символов отображаются □:** ваш шрифт не поддерживает эти символы. Попробуйте «Noto Serif», «DejaVu Sans» или «Times New Roman».

## Устранение неполадок

| Проблема | Решение |
|----------|---------|
| AltGr не работает | Выполните в терминале `setxkbmap -option lv3:ralt_switch` |
| Вместо символов □ | Используйте шрифт с поддержкой Unicode (Noto Serif, DejaVu Sans) |
| `setxkbmap` не работает | Возможно, вы используете Wayland. Настройте через Настройки → Клавиатура |
| Раскладка не видна на macOS | Выйдите из системы и войдите снова. Если не помогло — перезагрузите |
| Мёртвая клавиша не даёт результат | Нажмите AltGr+Z и **отпустите**, затем нажмите букву. Не удерживайте обе клавиши |

## Требования

- **Linux**: Ubuntu, Debian или производные (X11/Wayland + XKB + Python 3)
- **macOS**: macOS 10.9 или новее
